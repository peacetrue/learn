== 方位分布

方位按照九宫格划分：

* Vertical Direction（垂直方向）
. top(t)：上
. middle(m)：中
. bottom(b)：下

* Horizontal Direction（水平方向）
. left(l)：左
. center(c)：中
. right(r)：右

== 布局实现

下表记录了 3 种布局方式，名称使用方位的缩写表示（即上述括号中的字母）。

|===
|名称 |描述 |使用场景 |特点 |效果

|tmb
|上中下 布局
|主页
|header 位置固定，main 高度自适应，全屏滚动
|

|tm(lcr)b
|上中（左中右）下 布局
|功能页
|header 位置固定，footer 在底部，拥有 3 种变体
|

|tm(lc(mb)r)
|上中(左中(中下)右) 布局
|功能页
|header 位置固定，footer 在中间底部，拥有 2 种变体
|
|===

每块区域使用不同的背景色以示区别。
